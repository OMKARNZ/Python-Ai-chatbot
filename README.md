PYTHON CMD CHATBOT
A terminal-based conversational interface powered by Python and the Google GenAI API.

DESCRIPTION
This project provides a lightweight, command-line interface chatbot. It leverages Google's generative AI models to deliver intelligent responses directly inside the terminal, eliminating user interface overhead.

PREREQUISITES
Python 3.10 or higher
Google AI Studio API Key

INSTALLATION

1. Clone the repository:
   git clone https://github.com/OMKARNZ/Pyhton-Ai-chatbot-.git
   cd your-repo-name

2. Install the required dependency:
   pip install google-genai

3. Configure your API key environment variable:
   
   For Linux and macOS:
   export API_KEY="your_api_key_here"

   For Windows Command Prompt:
   set API_KEY="your_api_key_here"

   For Windows PowerShell:
   $env:API_KEY="your_api_key_here"

USAGE

Launch the chatbot by executing the main script:
python chatbot.py

To terminate the session, type exit or quit.
